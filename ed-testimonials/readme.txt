Ed Simple Testimonial Provide both widget and shortcode to help you display your testimonial page on your website.

== Description ==

Ed Simple Testimonial is an easy to use plugin that allows admin to add Testimonials to the sidebar, as a widget, or to embed them into a Page or Post using the shortcode. The Ed Simple Testimonial plugin also allows you to reorder the testimonial on the option Testtimonial > Reorder. Simple Testimonials allows you to include an image with each testimonial - this is a great feature for adding a photo of the testimonial author.

Features:

* Simple & Flat Design.
* HTML5 & CSS3.
* Using Custom Post Type - Testimonial.
* Shortcode for "Testimonial Slider".
* Testimonial Slider Widgets.
* Horizontal Slide Using Owl Carousel.


Shortcode :

* You can Add testimonial Author Name, in Admin site and Show the Testimonials List using below shortcodes :
* Default Shortcode [ed-testimonial].
//* If You Want to Hide Testimonial Photo, Add Shortcode in Page [ed-testimonial photo="no"]
* Use shortcode like this [testimonials show="3"] if how many testimonial slide you want to appear on testimonial slider.

== Installation ==

1. Upload the `ed-testimonials` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
